# NIT Manipur Portfolio

A responsive portfolio website for the National Institute of Technology (NIT) Manipur, built using **HTML5**, **CSS3**, and **Bootstrap 5**.

## Features

-   **Responsive Navbar**: Includes logo, navigation links, and a dropdown for branches.
-   **Image Carousel**: Highlights campus life and events.
-   **Department Progress Bars**: Visual representation of department growth.
-   **Contact Form**: Functional user interface for inquiries.
-   **Interactive Map**: Google Maps integration for location.
-   **FAQ Section**: Accordion-style Frequently Asked Questions.
-   **Footer**: Quick links, contact info, and copyright.

## Technologies Used

-   **HTML5**
-   **CSS3**
-   **Bootstrap 5.3.7**
-   **Bootstrap Icons**

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/dk2430098/college_portfolio.git
    ```
2.  Open `index.html` in your browser.

## Project Structure

-   `index.html`: Main HTML file.
-   `css/`: Contains custom styles (`style.css`).
-   `images/`: Contains project images.
